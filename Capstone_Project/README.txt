# System to predict stock prices

Include the '.SA' prefix before start the seaching:

p. ex.: PETR4.SA, ABEV3.SA

# Server Initialize:

Navegar até a pasta do arquivo 'manage.py'

executar no Shell: python manage.py runserver, para ativar o servidor interno

Acessar:

http://127.0.0.1:8000/index/

Para acessar a página de treinamento.
